Development Environment
=======================

Setup
-----

::

    # apt install python3.5 python3.5-venv
    $ make


Run
---

::

    $ make server


Browser
-------

::

    # apt install python3-pyqt5.qtwebkit
    $ make browser


Setup Labor API
---------------

::

    $ git clone https://github.com/freieslabor/labor-api
    $ cd labor-api
    $ make server
